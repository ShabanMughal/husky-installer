# Installation

## Using npx (Recommended)

The easiest way to use Husky Installer is with npx:

```bash
npx husky-installer
```

## Global Installation

Install globally to use the CLI anywhere:

```bash
npm install -g husky-installer
```

Then run:

```bash
husky-installer
```

## Using Bun

```bash
bun add -g husky-installer
```

## Requirements

- Node.js >= 18.0.0
- Git repository initialized
