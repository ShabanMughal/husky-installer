## Monorepo Setup

For projects with multiple packages:

<DocsCodeBlock
  title=".husky/pre-commit"
  lang="bash"
  code={`#!/usr/bin/env sh
. "$(dirname -- "$0")/_/husky.sh"

# Run checks in all packages
npm run lint --workspaces
npm run format --workspaces
npm run test --workspaces`}
/>

<DocsCodeBlock
  title="package.json (root)"
  lang="json"
  code={`{
  "name": "my-monorepo",
  "private": true,
  "workspaces": [
    "packages/*"
  ],
  "scripts": {
    "lint": "eslint .",
    "format": "prettier --write .",
    "test": "jest"
  }
}`}
/>

## CI/CD Integration

Example GitHub Actions workflow with Husky:

<DocsCodeBlock
  title=".github/workflows/ci.yml"
  lang="yaml"
  code={`name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  test:
    runs-on: ubuntu-latest
    
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: '18'
          
      - name: Install dependencies
        run: npm ci
        
      - name: Run linter
        run: npm run lint
        
      - name: Check formatting
        run: npm run format:check
        
      - name: Run tests
        run: npm test`}
/>

<Callout type="info" title="Note">
Husky hooks only run locally. In CI/CD, you need to run the checks explicitly as shown above.
</Callout>
