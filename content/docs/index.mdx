## Getting Started

The easiest way to get started is using `npx` (no installation required):

<PackageManagerTabs
  commands={{
    npm: 'npx husky-installer',
    pnpm: 'pnpm dlx husky-installer',
    yarn: 'yarn dlx husky-installer',
    bun: 'bunx husky-installer',
  }}
/>

This will guide you through an interactive setup process.

### Global Installation

Alternatively, you can install it globally:

<PackageManagerTabs
  commands={{
    npm: 'npm install -g husky-installer',
    pnpm: 'pnpm add -g husky-installer',
    yarn: 'yarn global add husky-installer',
    bun: 'bun add -g husky-installer',
  }}
/>

Then run it from any directory:

<PackageManagerTabs
  commands={{
    npm: 'husky-installer',
    pnpm: 'husky-installer',
    yarn: 'husky-installer',
    bun: 'husky-installer',
  }}
/>

> **Note:** Make sure you have initialized a git repository (`git init`) before running the installer.
